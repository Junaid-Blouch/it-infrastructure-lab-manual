## 📚 Overview

This repository contains the complete lab manual for the **IT Infrastructure** course. It covers fundamental to intermediate topics in computer networking, server configuration, virtualization, Linux shell scripting, and network monitoring.

All experiments are documented with objectives, required hardware/software, step‑by‑step procedures, and conclusions.
